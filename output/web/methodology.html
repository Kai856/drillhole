Methodology

Neural field prediction of subsurface geology in the Adavale Basin, Queensland, Australia.

1. Data Source

The input data is a 3D voxel grid exported from a GoCAD geological model of the Adavale Basin. The grid has dimensions 512 x 634 x 335 (108,584,480 voxels), with a spatial resolution of 500m x 500m x 20m (horizontal x vertical). The coordinate reference system is GDA94 / MGA Zone 55.

Each voxel contains an integer lithology code (1-13) identifying one of 13 geological formations, ranging from deep basement granite to shallow Cenozoic sediments. The data was loaded from a GoCAD .vop1 binary file (big-endian float32, Fortran column-major order).

2. Architecture

The model is a neural field (also called a neural implicit function): a function f(x, y, z) → lithology class that takes a 3D coordinate and predicts the geological formation at that location.

The architecture consists of:

  • Fourier positional encoding with 10 frequency bands. Each input coordinate is expanded into sine and cosine components at exponentially increasing frequencies (2^0 to 2^9), producing a 63-dimensional encoding from the 3 input coordinates. This enables the network to learn high-frequency spatial patterns.
  • 5-layer MLP with 256 hidden units per layer, using ReLU activations.
  • Skip connection at the mid-layer (layer 3), where the original positional encoding is concatenated back into the hidden representation. This helps preserve fine spatial detail through the network depth.
  • Classification head: a linear layer mapping from 256 to 13 output logits.
(x, y, z) → Positional Encoding (63-dim) →
Linear(63, 256) + ReLU → Linear(256, 256) + ReLU →
[skip: concat encoding] → Linear(319, 256) + ReLU →
Linear(256, 256) + ReLU → Linear(256, 13) → logits

3. Train / Validation Split

Instead of a random voxel-level split (which would leak spatial information), the model uses a drill-hole holdout strategy designed to simulate real-world prediction scenarios.

100 synthetic wells are randomly placed across the basin (seed=42). Each well is a 10x10 cell block (5km x 5km) extending through the entire depth of the model. All voxels within these well columns are held out from training — the model never sees any data from these locations.

This means the model must spatially interpolate to predict geology at the holdout locations, just as it would when predicting at a real undrilled location. This is a much harder test than random splitting, which would allow the model to memorize nearby voxels.

4. Training

The model is trained for 30 epochs using the following configuration:

  • Loss function: Cross-entropy with inverse-frequency class weights, ensuring rare formations (like Boree Salt) receive higher loss penalties.
  • Optimizer: Adam with initial learning rate 1e-3.
  • Scheduler: Cosine annealing (decays lr to ~0 over 30 epochs).
  • Batch size: 131,072 voxels per batch.
  • Epoch subsampling: Each epoch randomly samples 10M voxels from the 99M training set (not all voxels are seen each epoch). This dramatically speeds up training while maintaining convergence.
  • Validation subsampling: 2M voxels sampled from the 3.1M holdout set each epoch.
  • Coordinate normalization: All (x, y, z) coordinates are normalized to [-1, 1] before input.
  • Device: Apple MPS (Metal Performance Shaders).

5. Results

The model achieves 94.9% overall validation accuracy on the held-out drill holes. Performance varies by formation — large, spatially continuous formations like Glendower and Eromanga are predicted near-perfectly, while thin or spatially complex units like Boree Salt are harder.

6. Inference & Visualization

The best checkpoint is used to predict lithology for all 109M voxels in the original grid, producing a complete 3D predicted volume at full resolution. This volume is subsampled (every 10th voxel) and exported as JSON for the interactive web viewer.

The viewer renders 100 validation drill holes showing real lithology from the ground truth volume. Users can select rectangular areas on the basin and reveal the neural field's predictions in those regions, allowing direct visual comparison between known well data and model predictions.
